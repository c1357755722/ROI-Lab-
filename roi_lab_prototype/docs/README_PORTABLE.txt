ROI Lab Portable Web Package
============================

This package can run on a Windows computer without installing Python.

How to use
----------
1. Unzip the package to any folder.
2. Double-click `launch_offline.bat`.
3. Your default browser will open the ROI Lab page.

Notes
-----
- No internet connection is required.
- No Python environment is required for the offline package.
- Recommended browsers: Edge or Chrome.
- Import images from the page, draw ROIs, and export CSV directly.

Optional
--------
- If Python is available on the computer, you can also use
  `launch_roi_lab_tool.bat` to start the local server mode.
